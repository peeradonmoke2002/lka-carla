# **การใช้งานเครื่องมือปัญญาประดิษฐ์**

"รายวิชานี้อนุญาตและส่งเสริมให้ใช้เครื่องมือ AI เป็นผู้ช่วยในการสืบค้นข้อมูล เขียนโค้ดและแก้ไขบั๊ก (Debugging) แต่ นักศึกษาจะต้องเป็นผู้รับผิดชอบต่อโค้ดทุกบรรทัดที่อยู่ในโปรเจกต์ หากมีการตรวจพบว่าโค้ดส่วนใดเกิดข้อผิดพลาด หรือนักศึกษาไม่สามารถอธิบายการทำงานของโค้ดที่ AI สร้างขึ้นมาระหว่างการสอบนำเสนอ (Project Pitching) ได้ จะมีผลต่อคะแนนโดยตรง"

กลุ่มของข้าพเจ้าขอรับรองว่า ในการจัดทำโครงงานนี้ มีการใช้งานเครื่องมือ AI (เช่น ChatGPT, Gemini, GitHub Copilot, Claude ฯลฯ) เพื่อช่วยในการพัฒนา โดยมีรายละเอียดความโปร่งใสดังต่อไปนี้:

**1. เครื่องมือ AI ที่ใช้งาน (AI Tools Used):**

(เช่น ChatGPT (GPT-4), Gemini Advanced, GitHub Copilot)

**2. วัตถุประสงค์และขอบเขตการใช้งาน (Purpose & Extent of Use):** (ให้ระบุให้ชัดเจนว่าใช้ AI ช่วยในส่วนใดบ้าง)

- [ ] **สืบค้นข้อมูล / ออกแบบระบบ (Research):** (อธิบายเพิ่มเติม เช่น ใช้ AI ค้นหาข้อมูลอ้างอิง และเลือกใช้อย่างไร)

- [ ] **สร้างโครงร่างโปรเจค / โค้ดพื้นฐาน (Boilerplate):** (อธิบายเพิ่มเติม เช่น ใช้ AI ช่วยขึ้นโครงสร้างโฟลเดอร์ เบื้องต้น)

- [ ] **เขียนลอจิก / อัลกอริทึม (Logic Generation):** (อธิบายเพิ่มเติม เช่น ใช้ AI ช่วยเขียนโปรแกรม สำหรับดึงภาพจากกล้อง)

- [ ] **ค้นหาและแก้ไขข้อผิดพลาด (Debugging):** (อธิบายเพิ่มเติม เช่น ใช้ AI ช่วยวิเคราะห์สาเหตุของ Error OpenCV หรือบั๊กการเชื่อมต่อ Python)

- [ ] **การเขียนรายงาน (Documentation):** (อธิบายเพิ่มเติม เช่น ใช้ AI ช่วยเรียบเรียงไวยากรณ์ในบทคัดย่อ)

**3. การตรวจสอบและรับรองความถูกต้อง (Human Validation & Accountability):** (กลุ่มของข้าพเจ้าได้ทำการตรวจสอบโค้ดหรือเนื้อหาที่ AI สร้างขึ้นอย่างไร เพื่อให้มั่นใจว่าระบบทำงานได้จริงและปลอดภัย)

(ตัวอย่างการตอบ: "โค้ด API สำหรับรับค่าจากเซ็นเซอร์ที่ AI เจนเนอเรตมาในตอนแรกไม่รองรับการทำงานแบบ Real-time ทางกลุ่มจึงได้ทำการศึกษาเพิ่มเติมและแก้ไขโค้ดด้วยตนเองโดยเพิ่มระบบ Polling เข้าไป นอกจากนี้ได้ทดสอบยิงข้อมูลจริงจากบอร์ด ESP32 เพื่อยืนยันว่าข้อมูลลง Database ได้อย่างถูกต้อง"

"ทางกลุ่มใช้ GitHub Copilot ช่วยในการ Auto-complete โค้ด Python ของโปรแกรม และใช้ Gemini ในการช่วยหาวิธีแก้ Error ตอนที่เชื่อมต่อกับ Database เพื่อเก็บข้อมูลเว็บผ่าน API โดย Gemini แนะนำให้เช็ค Type Mismatch ของตัวแปรทศนิยม ซึ่งทางกลุ่มได้นำมาวิเคราะห์และปรับแก้คอลัมน์ใน Database เป็น Float ด้วยตนเองจนระบบทำงานได้")
